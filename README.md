# Smart Dustbin Monitoring System using ESP32

## Overview
This project monitors the fill level of a dustbin using an ultrasonic sensor and ESP32. When the dustbin reaches a predefined level, an alert is generated.

## Features
- Fill Level Detection
- Full Bin Alert
- Real-Time Monitoring
- Smart Waste Management

## Components
- ESP32
- HC-SR04 Ultrasonic Sensor
- LED
- Breadboard

## Working
The ultrasonic sensor continuously measures the distance between the sensor and the waste. If the distance falls below a threshold, the system indicates that the dustbin is full.
